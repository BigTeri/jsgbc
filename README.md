## Controls

> Keyboard **keys**
  - A (**l**, **v**)
  - B (**k**, **x**)
  - START (**enter**)
  - SELECT (**shift**)
  - ↑ (**up**, **w**)
  - ↓ (**down**, **s**)
  - ← (**left**, **a**)
  - → (**right**, **d**)
  - SPEED (**p**)
> Gamepad [**button index**](https://www.w3.org/TR/gamepad/#remapping)
  - A (**1**)
  - B (**0**)
  - START (**9**)
  - SELECT (**8**)
  - ↑ (**12**)
  - ↓ (**13**)
  - ← (**14**)
  - → (**15**)
  - SPEED (**7**)
> Software Buttons from **[jsGBC-ui](https://github.com/ardean/jsGBC-ui/)**

## Features

> Screen

- Pixelated
- Fullscreen Mode

> Files

- ROM Upload (**.gb** / **.gbc**)
- Battery File Download / Upload (**.sav**)
